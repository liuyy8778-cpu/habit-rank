⑤ 本頁對照的紅線(自檢)

只長不枯:樹身讀 maxTier 高水位、果實永久掛樹、花只開不謝 → 全頁無任何凋謝/枯萎/褪色呈現
不做 streak / 每日壓力:無進度條、無百分比、無連續天數、無「還差 N」→ 三態純離散 ✓
零產出:此頁純展示,不給金幣、不解鎖任何東西 ✓
不綁信任現況:樹只講「曾經到過哪」,信任降級不落回花期 ✓
視覺語言統一:SVG 只做樹幹/枝條,花果 = emoji(同②家園) ✓
果實統一(拍板 D 第6條):全樹果實一律 🍎,禁品種/稀有度/圖鑑式差異 → 樹是紀念冊非圖鑑 ✓
構圖留白(布局補充):樹體偏軸佔 ~55% 寬,右側留白為外觀槽位預留(本分支不做槽位) ✓
動畫紅線(此頁預留,尚未實作):僅允許進頁一次性生長回放 ≤2s + 新果 popIn 一次 + floaty 微搖曳; 禁進行式/粒子/閃爍/循環變色
